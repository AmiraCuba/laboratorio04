Multiplicar dos numeros en Java

  1. Importa la clase java.util.Scanner y crea el objeto: Scanner sc = new Scanner(System.in);
  2. Crear las variables de tipo entero con: int numero1, numero2, producto
  3. Solicitar al usuario el valor para numero1 con: numero1 = sc.nextInt();
  4. Solicitar al usuario el valor para numero2 con: numero2 = sc.nextInt();
  5. Darle el valor a producto de la multiplicacion de numero1 y numero2 con: producto = numero1*numero2;
  6. Mostrar el resultado en pantalla con: System.out.pritn(producto);
